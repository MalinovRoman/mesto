#### Сайт адаптирован под различные девайсы и создан с помощью HTML и CSS, в том числе адаптивность достигается применением Flexbox и Grid. Файловая структура создана с пременением Nested, название блоков, элементов и модификаторов создано по БЭМ. Часть логики написана на JavaScript.
